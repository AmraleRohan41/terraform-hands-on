## Overview

This repository contains hands-on Terraform projects and Infrastructure as Code (IaC) implementations using Terraform and AWS Cloud.

The project demonstrates how to automate cloud infrastructure provisioning, manage resources efficiently, and follow Terraform best practices.

---

## Technologies Used

- Terraform
- AWS Cloud
- Infrastructure as Code (IaC)
- Git & GitHub
- Linux

---

## Features

- AWS provider configuration
- Terraform state management
- Infrastructure automation
- Variables and outputs
- Resource dependency management
- Reusable Terraform configuration
- Infrastructure provisioning using code
